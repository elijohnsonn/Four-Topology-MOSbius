# Overview
Repository for 2026 SSCS Chipathon Four Topology MOSbius team

Configurable chip offering comparison of four single-ended op-amp topologies implemented in GF180mcuD:
- Common-Source (`core_amps/cs_stage`)
- 5-Transistor OTA (`core_amps/ota_5t`)
- Telescopic Cascode OTA (`core_amps/ota_telescopic`)
- Folded Cascode OTA (`core_amps/ota_folded_cascode`)
